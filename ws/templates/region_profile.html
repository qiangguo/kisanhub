Before you start simulating the data collection and analysis, you will need to specify the regional data resources and save them into database. The recommended resources have been included in this page. Please copy regional names such as UK, Wales, ... and their URLs into the corresponding fields. The data are classified into to present types, i.e., RANKED and ORDERED. Please make sure you have specified the correct resource present type. Wrongly selected resource present type leads to failure upon data parsing.

There exist abbreviation definions for the regional fields, these being:

  • MAX : Max Temp
  • MIN : Min Temp
  • MEN : Mean Temp
  • SSN : Shunshine
  • RNF : Rainfall
  • RND : Raindays
  • DAF : Days of Air frost
